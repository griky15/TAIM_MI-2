## Overview

This repository contains a small interactive prototype for a gestural instrument simulator (Leap Motion + mouse-based prototype). The main demo is an HTML/WebAudio-based prototype that maps hand/mouse position to pitch, rhythm and volume.
l_prototype/prototype2.html` in your browser.

## Controls

- Mouse X (horizontal): change note (along the selected scale)
- Mouse Y (vertical): change rhythm subdivision (faster/slower note rate)
- Mouse wheel: change depth/volume
- Keys 1-5: select musical scale (1 = C Major, 2 = C Minor, 3 = Pentatonic, 4 = Blues, 5 = Chromatic)
- Arrow ← / → : change instrument (Synth → Guitar → Saxophone → Piano)
- Arrow ↑ / ↓ : increase / decrease BPM (±5 BPM)
- Start Audio button: start / stop audio engine
- Info (i) button: toggle on-screen instructions
